## To Set Up Locally:

1. **Clone the GitHub Repo:**
    ```bash
    git clone https://github.com/Avichiii/PythonWebAPP
    ```

2. **Run `app.py` File:**
    ```bash
    python app.py
    ```

3. **Navigate to the Browser:**
    Open your web browser and go to [http://127.0.0.1:<port>](http://127.0.0.1:<port>)
